# FlexSeat - Telescopic Outdoor Stool Website

This is a React-based website for **FlexSeat**, a portable telescopic outdoor stool. The website is designed as a product landing page, highlighting the features, benefits, and purchase options for the product.

## Features

- Responsive design for desktop and mobile
- Hero section with main product message
- Features section showcasing key benefits:
  - Ultra portable
  - Heavy-duty and durable
  - Adjustable height
- About section describing the product and brand
- Buy Now section with call-to-action button
- Footer with copyright info

## Technologies Used

- React
- Tailwind CSS (for styling)
- JavaScript / JSX

## Usage

1. Clone this repository:
```bash
git clone https://github.com/yourusername/flexseat-website.git
```

2. Navigate to the project directory:
```bash
cd flexseat-website
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

5. Open [http://localhost:3000](http://localhost:3000) to view the website in your browser.
